## LaTeX 入门教程

本仓库包含一份面向**数学系本科二年级学生**的 LaTeX 入门教程，源文件为 [`latex_tutorial.tex`](latex_tutorial.tex)。

### 教程内容

| 章节 | 内容 |
|------|------|
| 第一部分 | 为什么要学习使用 LaTeX |
| 第二部分 | 什么是 LaTeX |
| 第三部分 | 在哪里使用 LaTeX（Overleaf 快速上手） |
| **第四部分** | **必知必会的命令与技巧（重点）** |
| 第五部分 | AI 时代如何高效使用 LaTeX |

### 使用方法

1. 访问 [Overleaf](https://www.overleaf.com) 并登录账号。
2. 点击 **New Project** → **Upload Project**，上传 `latex_tutorial.tex`，  
   或新建空白项目后将文件内容粘贴到 `main.tex`。
3. 在 Overleaf 左上角 **Menu** → **Compiler** 选择 **XeLaTeX**（支持中文）。
4. 点击 **Recompile** 编译，即可在右侧看到排版好的 PDF 教程。
