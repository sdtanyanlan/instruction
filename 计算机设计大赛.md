# 参加中国大学生计算机设计大赛（4C）分析报告

> **参考赛事**：2025 年（第 18 届）中国大学生计算机设计大赛（官网：https://jsjds.blcu.edu.cn）

## 一、可参加的赛道方向

本项目横跨 AI 技术、教育数字化与数学可视化三大领域，**高度契合**以下三条赛道，建议优先选择其中 1-2 条参赛：

### 🏆 首选赛道一：微课与 AI 辅助教学（强烈推荐）

| 项目 | 说明 |
|------|------|
| **方向匹配度** | ⭐⭐⭐⭐⭐ 极高 |
| **国赛赛区** | 南京南赛区 |
| **作品形式** | AI 辅助生成的教学微课 + 系统平台 |
| **核心要求** | 体现 AI 在教学流程中的融合度、智能化特色、课程效果可验证 |

**匹配理由：**
- 本项目以 LLM+Manim 自动生成数学教学动画，天然属于"AI 辅助教学内容生成"场景；
- 极限 ε-δ、线性变换、中心极限定理三个典型案例可直接作为参赛微课作品；
- 有完整的准实验数据支撑教学效果（概念理解 +14.9%、兴趣 +15%、备课效率 ×15），是其他参赛作品难以比拟的优势。

## 二、核心竞争优势分析

### 2.1 技术创新优势

| 优势点 | 具体说明 |
|--------|---------|
| **LLM × Manim 深度融合** | 将自然语言处理与数学动画引擎打通，不是简单调 API，而是完整的意图识别→脚本生成→代码生成→渲染管线 |
| **自动纠错迭代机制** | "生成—校验—修正"三轮迭代策略解决了 LLM 代码生成错误率高的实际问题，体现工程落地能力 |
| **Prompt 模板体系** | 面向数学教学的结构化 Prompt 设计，具有较强的可复现性和可扩展性 |
| **代码模板库** | 覆盖高数、线代、概率论三大课程的 6 类模板，工程完整度高 |

### 2.2 教育应用优势

| 优势点 | 具体说明 |
|--------|---------|
| **真实实验数据** | 准实验设计，200 人规模（98 实验组 vs 102 对照组），一学期周期，数据可信度高 |
| **显著提升效果** | 概念理解 +14.9%（p<0.01）、学习兴趣 +15%、备课效率提升约 15 倍，数据亮眼 |
| **三个完整案例** | ε-δ 极限、线性变换、中心极限定理，覆盖三门核心数学课，案例丰富 |
| **教育公平价值** | 为偏远地区教师提供与名校同等质量的可视化资源，社会意义突出 |

### 2.3 理论创新优势

| 优势点 | 具体说明 |
|--------|---------|
| **"内容即服务（CaaS）"概念** | 将云计算 SaaS/PaaS 理念引入教育领域，概念新颖，有学术深度 |
| **范式转变框架** | "资源建设"→"能力建设"的转变，有政策对接（国家教育数字化战略行动） |
| **文献综述完整** | 覆盖 UNESCO 报告、国家政策文件、LLM 技术报告等 15 篇参考文献，学术基础扎实 |

## 三、参赛策略建议

### 3.1 最优参赛路径（推荐）

```
主赛道：微课与 AI 辅助教学
↓
以"AI 驱动的数学微课按需生成平台"为核心作品
↓
提交内容：
  ① 3 个完整微课视频（ε-δ极限 / 线性变换 / 中心极限定理）
  ② Web 平台演示（教师输入→动画生成完整流程录屏）
  ③ 研究报告/说明书（可基于现有论文精简）
  ④ 教学效果数据报告（实验数据可视化）
```

### 3.2 作品打磨重点

**第一优先级（决定能否获奖）：**
- [ ] 制作 3-5 分钟完整演示视频，展示"教师输入→AI处理→动画输出"全流程
- [ ] 将系统做成可在线访问的 Demo（哪怕是录屏视频也比纯 PPT 有力）
- [ ] 制作数据可视化图表（实验组 vs 对照组对比图）

**第二优先级（提升竞争力）：**
- [ ] 优化 Web 平台 UI/UX，使教师操作更直观友好
- [ ] 增加"Prompt 模板市场"功能，体现社区共享理念
- [ ] 准备英文摘要（部分赛区评审有国际评委）

**第三优先级（加分项）：**
- [ ] 争取与合作学校签署试用协议，作为真实应用场景证明
- [ ] 整理代码并开源到 GitHub，提升项目可信度和技术展示

### 3.3 潜在风险与应对

| 风险点 | 应对策略 |
|--------|---------|
| AI 生成代码错误率较高（约首次成功率 60-70%） | 在演示中突出展示自动纠错机制，将"挑战"变为技术亮点 |
| 实验数据为初稿示例性数据 | **务必在参赛前替换为真实实验数据**，否则有学术不端风险 |
| 参赛作品需可运行演示 | 提前准备录屏视频作为备用，避免现场演示故障 |
| 同类竞争作品（AI+教育赛道热门） | 突出 Manim 数学动画的专业性和准实验设计的严谨性，形成差异化竞争 |

## 五、总结

本项目在**微课与 AI 辅助教学**赛道具有极强的竞争力，是三条赛道中**最优选择**。核心优势在于：

1. **技术路径清晰**：LLM + Manim 的完整管线在国内竞赛中较为罕见；
2. **教育效果有据**：有准实验数据支撑，是大多数参赛作品缺乏的量化验证；
3. **社会价值突出**：教育公平、教师减负两大主题契合国家政策导向；
4. **成果形态完整**：论文 + 系统 + 案例视频，参赛材料天然完备。

**建议行动**：在现有论文基础上，尽快完成系统 Demo 的开发和演示视频的录制，即可形成完整的参赛作品包。
